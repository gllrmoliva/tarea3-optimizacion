# tarea3-optimizacion
Tarea que consiste en implementar el modelo matemático que resuelve el problema de job shop flexible, propuesto por Fattahi et al. 2007. 

## Requisitos
- Utilizar versión 3.13.7 de python (se puede utilizar conda o pyenv)
- Tener instalado previamente gurobi optimizer con su respectiva licencia.
- Instalar librerias necesarias con `pip install -r requirements.txt`

## Ejecución
Dentro del directorio principal (\<DIR\>/tarea3-optimizacion/) ejecutar:
``` bash
python main.py
```

> Instancias de problema obtenidas de: https://github.com/SchedulingLab/fjsp-instances/
